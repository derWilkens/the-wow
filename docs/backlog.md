[] wow-edge-data-stack: bei mouse over erscheint direkt ein popup mit der Liste der Datenobjekte und leicht verzögert ein weiteres PopUp. das zweite entfernen
[] wow-edge-data-chip: mouse over popup enfernen, bei klick muss der Detaildialog im Vordergrund angezeigt, der Button "Details der Verbindung" entfernen, bei klick außerhalb des Dialogs soll dieser geschlossen werden
[] Start-Knoten, Zielknoten: die Knotengröße ist nicht kompatibel mit den Snap Grid Abstand. Das hat zur Folge, dass die Linie vom Startknoten zu einer Aktivität einen Sprung hat.
[] Toolbar: beim Mouse over über ein Werkzeug, soll der Name nur als popover angezeigt werden.
[] Wenn ein Datenspeicher eingefügt wird, soll der Benutzer die Möglichkeit haben, einen bereits bestehenden zu wählen
[] Aktivitätsknoten: wenn der Knoten markiert ist, schneidet der Rahmen die Connectoren ab, diese sollen vollständig angezeigt werden.
[] Beim Anlegen einer Aktivität soll die Beschreibung leer sein, keine default text.
[] Der Detaildialog soll automatisch speichern und geschlossen werden, wenn außerhalb des Dialogs geklickt wird.Wenn der Detaildialog geöffnet ist und eine andere Aktivitätsknoten selektiert wird, sollen die Details dieses Knoten angezeigt werden.
[] wenn eine Konten gezogen wird und dabei mit einem anderen kollidiert, soll der andere Knoten ausweichen, so wie hier beschrieben: https://xyflow.com/blog/node-collision-detection-algorithms diese Funktion kann der Benutzer in den Settings deaktivieren, per default ist die funktion aktiv
